# 🧂 Salt'n Pepper — Website Clone

A clean and pixel-perfect front-end clone of the official [Salt'n Pepper Restaurants](https://www.saltnpepper.com.pk/) website — one of Pakistan's most iconic restaurant chains. Built as a CSS assignment using pure HTML and CSS, replicating the restaurant's UI, layout, and branding.

## ✨ Features

- 🏠 Hero / Landing section with banner slider UI
- 🍽️ Brand showcase section (Salt'n Pepper Restaurant, Express, Village, Signature)
- 📍 Franchise locations listing with contact details
- 🛒 Order Online CTA buttons
- 🧭 Responsive navigation bar with logo and links
- 📋 About Us, Publications, Feedback, and Franchise page links
- 📱 App store badges (Google Play & Apple Store)
- 🔗 Footer with Site Links, Contact Info, and social media icons
- 🌐 Built with pure HTML & CSS — no frameworks or libraries

## 🚀 Getting Started

### Run Locally

1. **Clone the repository**

```bash
git clone https://github.com/Umairalimalik78/Salt-n-Pepper-Clone.git
```

2. **Navigate to the project folder**

```bash
cd Salt-n-Pepper-Clone
```

3. **Open `index.html` in your browser**

Simply double-click the `index.html` file or open it with **Live Server** in VS Code.

> No build tools, frameworks, or installations required — it's pure HTML & CSS!
